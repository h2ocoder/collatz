# Stopping Point Line Conjecture
**Source:** Paper 2, Conjecture 1

> All [[Stopping Point]]s belonging to $\text{Sclass}_k$ fall on one of $\text{Smod}_k$ lines, without exception.

**Status:** Empirically verified.

**Equations for known classes:**
- Class 3: $3x + y - 1 = 0$
- Class 6: $9x + 7y - 5 = 0$
- Class 8 (offset 0): $27x + 5y - 23 = 0$
- Class 8 (offset 1): $27x + 5y - 19 = 0$
