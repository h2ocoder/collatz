# Dropping Destination
**Source:** Paper 1, Definition 4

The first number reached via iteration of the Collatz function that is less than $n$.

**Notation:** $\text{Ddest}_n$

**Example:** $\text{Ddest}_3 = 2$, $\text{Ddest}_5 = 4$

**Related:** [[Dropping Time]], [[Dropping Orbit]]

**Code:** `collatz.dropping.dropping_destination(n)`
