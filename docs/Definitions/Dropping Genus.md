# Dropping Genus
**Source:** Paper 1, Definition 10

A unique tuple $(\text{Set}, \text{Modulus}, \text{Index})$ that completely classifies each integer's dropping behavior.

**Examples:**
- $n = 3$: Genus = $(6, 0, 0)$
- $n = 15$: Genus = $(11, 1, 0)$
- $n = 635$: Genus = $(13, 3, 2)$
- $n = 5183$: Genus = $(19, 7, 1)$
- $n = 27$: Genus = $(96, 0, 0)$

**Related:** [[Dropping Set]], [[Dropping Modulus]], [[Dropping Index]], [[Stopping Signature]]

**Code:** `collatz.dropping.dropping_genus(n)`
