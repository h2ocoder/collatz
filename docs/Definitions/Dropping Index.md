# Dropping Index
**Source:** Paper 1, Definition 9

The 0-based position of $n$ within its [[Dropping Set]] and [[Dropping Modulus]], when ordered by magnitude.

**Example:** In $\text{Dset}_6$, modulus 0: $n=3$ has index 0, $n=19$ has index 1.

**Related:** [[Dropping Genus]]

**Code:** `collatz.dropping.dropping_index(n)`
