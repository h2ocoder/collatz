# Dropping Set
**Source:** Paper 1, Definition 6

$\text{Dset}_k$ is the ordered set of integers requiring exactly $k$ iterations to drop below their starting value.

- $\text{Dset}_1$ = all even numbers ($a_n = 2n$)
- $\text{Dset}_3$ = numbers $\equiv 1 \pmod{4}$ ($a_n = 4n+1$)
- $\text{Dset}_6$ = generated by $a_n = 16n - 3$
