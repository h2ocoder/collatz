# Orbital Oddity
**Source:** Paper 1, Definition 7

The count of odd integers within a [[Dropping Orbit]].

| Dropping Set | Orbital Oddity |
|-------------|---------------|
| 1 | 0 |
| 3 | 1 |
| 6 | 2 |
| 8 | 3 |
| 11 | 4 |
| 13 | 5 |
| 16 | 6 |

**Conjecture 1:** All Dropping Orbits in $\text{Dset}_k$ share the same Orbital Oddity. See [[Orbital Oddity Conjecture]].

**Code:** `collatz.dropping.orbital_oddity(n)`
