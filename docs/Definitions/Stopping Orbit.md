# Stopping Orbit
**Source:** Paper 2, Section 3

The sequence from $f(n)$ to the [[Stopping Destination]]. Excludes $n$, includes destination.

**Notation:** $\text{Sorbit}_n$

**Example:** $\text{Sorbit}_{19} = [58, 29, 88, 44, 22, 11]$

Contrast with [[Dropping Orbit]] which includes $n$ and excludes destination.

**Code:** `collatz.stopping.stopping_orbit(n)`
