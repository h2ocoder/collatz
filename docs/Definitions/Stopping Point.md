# Stopping Point
**Source:** Paper 2, Section 3

A point $(x, y)$ derived from the stopping behavior:
- $x = \text{Sdest}_n - n$
- $y = \text{Sdest}_n$

**Example:** $\text{Spoint}_{19} = (-8, 11)$

Points within the same [[Stopping Class]] fall on linear Diophantine lines.

**Code:** `collatz.stopping.stopping_point(n)`
