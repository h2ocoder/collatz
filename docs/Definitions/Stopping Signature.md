# Stopping Signature
**Source:** Paper 2, Section 3

A unique tuple $(\text{Time}, \text{Page}, \text{Offset})$ identifying an orbit's location in [[Collatz Address Space]].

**Examples for Class 8:**
| n | Signature |
|---|-----------|
| 11 | (8, 1, 0) |
| 23 | (8, 1, 1) |
| 43 | (8, 2, 0) |
| 55 | (8, 2, 1) |

**Related:** [[Stopping Time]], [[Stopping Page]], [[Stopping Offset]], [[Dropping Genus]]

**Code:** `collatz.stopping.stopping_signature(n)`
