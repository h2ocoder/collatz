# Stopping Time
**Source:** Paper 2, Section 2

The number of steps needed to arrive at a number lower than the starting number $n$.

Equivalent to [[Dropping Time]].

**Notation:** $\text{stop}_n$

**Example:** $\text{stop}_5 = 3$, $\text{stop}_{19} = 6$, $\text{stop}_{27} = 96$

**Code:** `collatz.stopping.stopping_time(n)`
