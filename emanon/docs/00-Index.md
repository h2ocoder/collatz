# Emanon

A sci-fi 4X multiverse game where physics emerges from information theory. Universes are prime-addressed event logs. Gravity is entropy leakage. Stars are compression engines. Governance is a game mechanic.

**Name origin:** "No name" reversed. The shared agreement to label the rules without claiming to be the rules.
