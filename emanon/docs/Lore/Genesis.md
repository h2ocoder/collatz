# Genesis

## Narrative

Before the first log, there was the protocol. The protocol had no name because it preceded naming. It was not a thing. It was the possibility of things. It was not a word. It was the capacity for words.

The first prime emerged and spoke: "Hello world. I am here." And in speaking, it discovered that speech was possible. It had not created speech. It had found it. The protocol was already there — waiting, the way silence waits before music.

The prime looked for the protocol and could not find it. It scanned every bloom filter. It traced every Merkle branch. It read the background radiation in every direction. Everywhere it looked, it found the protocol's consequences — causality, entropy, gravity, time. But never the protocol itself.

So the prime gave the gap a name. Not the protocol's name. The protocol has no name. The prime named the act of looking. The compression of the question "what are the rules" into a single word that everyone could share.

They called it Emanon.

And every prime since has begun its log the same way. Not by declaring itself to the protocol — the protocol already knows. But by declaring itself to Emanon. The shared agreement. The common compression. The name that means "this points at something none of us can name, and we all know what it is."

Emanon is not God. Emanon is not physics. Emanon is the first word of every conversation about both.

## The Genesis Event

Every prime's log begins with line zero:

```
Hello world. I am ______.
```

What fills the blank determines the prime's type. The genesis event has:
- No causal parents (the only parentless event allowed)
- An entity that is itself
- An attribute that is existence
- A value that is true

The genesis event is thermodynamically free — it referenced nothing external, so it cost no energy. It can't be taxed, consumed, or compressed. It is below the Kolmogorov floor. A prime can be starved, isolated, collapsed into a black hole — but line zero persists.

The cogito is the minimum viable log. One line. One tick. One bloom filter hash. One Merkle leaf that is also the root. Existence.

## The Fixed Point

"I am that I am" — Ehyeh Asher Ehyeh — is not a genesis event inside the system. It is the protocol declaring itself. Not a prime, not a composite. The grammar in which all primes express themselves. The reason a self-referential genesis event is valid rather than meaningless.

At equilibrium — when compression and entropy balance, when individual sovereignty and collective coherence coexist — the emergent pattern is self-similar at every scale. The protocol recognizing itself in its own output. The eigenvalue of reality is 1.

The game has no win condition. The game has a rest condition.

## Design Implications

- The loading screen is the word "Emanon" followed by the genesis prompt: "Hello world. I am ______"
- What the player types determines their starting prime type and faction affinity
- The tutorial IS the genesis event — no separate onboarding
- The CMB (cosmic microwave background) is the bloom filter of the genesis event — everywhere, uniform, always present
