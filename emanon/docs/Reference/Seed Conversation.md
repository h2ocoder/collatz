1. **Multiverse as distributed VCS** — each universe is a branch, observation events are commits, causality is the DAG, speed of light limits edge formation rate

2. **Event sourcing** — universe state is derived, not stored; a universe is a log of events; current state = fold over log up to time t

3. **Prime factorization addressing** — each prime is an atomic stream; composite address = product of primes = subscription list; GCD = shared state; LCM = merge

4. **Causal threads as atomic unit** — not entities, not attributes; an irreducible chain of cause-and-effect; a worldline

5. **LLM-backed primes** — each stream is an agent with tool calls (observe, emit, entangle); the system prompt is the local physics

6. **4X mapping** — Explore = observe(); Expand = mint primes; Exploit = fold streams; Exterminate = forced merge (hostile observation)

7. **Protocol = physics, prompt = culture** — the tool call API is non-negotiable gravity; the LLM prompt is local and moddable

8. **Individual-collective axis** — conflicts are political events; anarchic/constitutional/authoritarian composites; governance as game mechanic

9. **Information-theoretic physics** — time = line count; mass = line length; entropy = compressibility; E=mc^2 emerges

10. **Gravity as entropy signal** — involuntary observation; can't be massive and hidden; dark matter = well-governed invisible empires

11. **Bloom filters as spectrum** — cheap probabilistic scanning; false positive rate IS the physics; filter saturation = gravitational noise

12. **Proof of work as exploration** — mining = factoring random addresses, scanning bloom filters; the blockchain is an atlas

13. **Energy = compression differential** — stars are compression engines; fuel = redundant logs; conservation = total info constant

14. **Merkle trees as timeline skeleton** — roots in bloom filters; timeline-level scanning; stars build trees as compression output

15. **Distance to impact** — signed causal metric; negative = prophecy (risky leverage); positive = normal causality

16. **Genesis and the cogito** — line zero is "I am"; thermodynamically free; eternal; below Kolmogorov floor

17. **"I am that I am" as fixed point** — not a prime, the protocol itself; the type system; discoverable at equilibrium

18. **Six log types** — particle, force, field, law, story, witness ("here")

19. **Forgiveness as mutual recomputation** — both parties lose mass; thermodynamically favorable long-term

20. **Emanon** — "no name" reversed; the shared compression; the first word of every conversation about the rules

21. **Em language** — causality is syntactic; energy is a linear type; compiler is a physicist; BEAM/Elixir as phase 2 target
